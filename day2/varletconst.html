Differences between var let and const variables


        VAR:
        i.)var variables are global scope variables.
        ii.)var variables can be hoisted.
        ii.)var variables can be redeclared.
        iii.)var variables can be updated.
        iv.)var variables once declared,can be initialised later also.

        

        LET:
        i.)let variables are block  scope variables.
        ii.)let variables can be hoisted,but they are hoisted to the temporal dead zone.
        ii.)let variables cannot be redeclared.
        iii.)let variables can be updated.
        iv.)let variables once declared,can be initialised later also.

        

        CONST:
        i.)const variables are block  scope variables.
        ii.)const variables can be hoisted,but they are hoisted to the temporal dead zone.
        ii.)const variables cannot be redeclared.
        iii.)const variables cannot be updated.
        iv.)const variables should be declared and initialised at the same time.

        

        

        HOISTING:Hoisting is a default behaviour of javascript to push the decalrations to the top.
        

        TEMPORAL DEAD ZONE: When let and const variables are declared ,they are said to be  in temporal dead zone until unless they get initialised with a value.
